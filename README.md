![screen](imguix.png)

## How to compile

* Clone this repository
* Create a new Cocos2D-X C++ project
* Copy and replace the contents of this repository into the new project
* Compile and run:
* Linux: `cocos run -p linux`
* Mac: `cocos run -p mac`
* Windows: `cocos run -p win32`
